# SOS 旁白录音对照表

> 发给疗愈师的录音指南。共 30 条音频，按阶段分组。

---

## 录音要求

- **语调**：温柔、安静、不急促，像一个陪伴在旁边的朋友
- **停顿**：`/` = 换行停顿约 1 秒，`……` = 拉长约 1.5–2 秒
- **格式**：.m4a 或 .mp3，44.1kHz，单声道，无背景噪音
- **音量**：尽量保持统一响度（后期会做 -16 LUFS 标准化，但原始录音差异越小越好）
- **命名**：请严格按照「文件名」列命名，扩展名用 .m4a
